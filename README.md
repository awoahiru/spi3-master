# SPI3 マスター

令和8年度 群馬県職員採用試験（社会人経験者）第1次試験（SPI3・テストセンター方式）対策アプリ。
書籍**「これが本当のSPI3だ！ 2028年度版」（講談社）**を3周して仕上げる学習管理＋反復ドリルツールです。

## 使い方

### Macで使う
`start.command` をダブルクリック → ブラウザが自動で開きます。
（初回に「開発元が未確認」と出たら、右クリック→「開く」）

### iPhoneで使う
1. Macで `start.command` を起動したままにする
2. iPhoneを**同じWi-Fi**に接続し、Safariでターミナルに表示された `http://<MacのIP>:8355` を開く
3. 共有ボタン →「**ホーム画面に追加**」→ 以後はアイコンからネイティブアプリのように起動

### MacとiPhoneの記録を同期する
「その他」タブ →「端末間の同期」で、片方の端末で**書き出し** → もう片方で貼り付けて**取り込み**。
（記録は端末のブラウザ内に保存されるため、自動同期はされません）

## 機能

- **ホーム** — 試験日カウントダウン、今日のタスク、連続学習ストリーク、進捗リング
- **プラン** — 開始日〜試験前日まで「1周目（理解）→2周目（速度）→3周目（弱点・実戦）→直前調整」を自動生成。毎日約2時間、日曜は予備日。試験日を変えると自動で組み直し
- **ドリル** — テストセンター形式の制限時間つきオリジナル問題88問（書籍の転載なし）。おまかせ/弱点特訓/模試/分野別。間違えた問題は復習キュー（間隔反復）へ
- **記録** — 分野別正答率、アクティビティヒートマップ、XP
- **その他** — 試験日設定、同期、テストセンター当日ガイド、性格検査のポイント

## 構成

```
index.html      画面
css/app.css     iOS風スタイル
js/data.js      書籍の章構成・プラン・オリジナル問題88問
js/app.js       アプリロジック
icons/          アプリアイコン
start.command   起動スクリプト（ポート8355）
```
